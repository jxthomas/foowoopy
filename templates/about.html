Why Foo Woo?

Foo Woo uses the New York City Restaurant Inspection data stored in a MongoDB database. The backend server is powered by Python using the Flask Framework. The front end is coded in AngularJS. The inspection data is enriched by REST calls using the Yelp API. The purpose of this demo application is to showcase our full stack programming offerings.
